手順:
1. **Claude サブエージェントを2つ並列に spawn**（**model は opus を明示指定**。セッションモデルを継承させない）し、各自に web 検索・bash を使って独立に同じ問いを解かせる（互いの結果は見せない）。
2. `~/.claude/skills/quorum/references/judge_rubric.md` に沿って、メイン（セッションのモデル）が2つの回答を突き合わせる。
3. 最終回答＋監査証跡（合意・矛盾・盲点）を出す。

> このモードは Claude Code のプラン内で動き、追加の従量課金は発生しない。
